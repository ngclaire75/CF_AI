What each agent does
infoScans open ports, server software, DNS records, and SSL certificates to map the target. confChecks for exposed config files, admin panels, and insecure server settings. jsReads JavaScript files to find leaked API keys, secrets, and hidden endpoints. idntTests how the site handles user accounts, registration, and identity verification.
athnTests login forms for weak passwords, missing lockout, and brute-force gaps. athzChecks if users can access pages or data they are not allowed to see. sessTests how login sessions and cookies are issued, stored, and expired. inpvTries SQL injection, XSS, command injection, and other input attacks. crypChecks SSL/TLS encryption strength, certificate validity, and cipher choices.
clntTests browser-side code and client-facing scripts for vulnerabilities. apitTests API endpoints for missing auth, data leaks, and injection flaws. enumFinds hidden API routes and tests for access control gaps across endpoints. ctfSolves Capture The Flag security challenges using automated attack chains. pentestRuns all 10 checks in sequence for a complete end-to-end security assessment.
