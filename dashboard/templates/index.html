Meet Our Agents
CyberINK deploys 11 specialised security testing agents, each mapped to the OWASP Web Security Testing Guide (WSTG). Every agent runs real shell commands — no hardcoded results.
Recon Scout
Passive & Active Reconnaissance
Performs deep passive intelligence gathering before touching the target. Runs WHOIS, DNS enumeration, certificate transparency, Wayback Machine, Shodan, HackerTarget, URLScan, AlienVault OTX, and subdomain discovery. Then moves to active HTTP fingerprinting via a 9-layer Cloudflare bypass cascade.

INFO-02 Server FingerprintINFO-06 Entry Points INFO-07 JS Bundle AnalysisINFO-08 Framework Headers INFO-09 App FingerprintINFO-10 Architecture Map Port ScanService Detection Dir Brute-forceNuclei Tech Scan
JS Secret Hunter
JavaScript & Source Code Analysis
Fetches every JS bundle referenced on the target page and scans for hardcoded secrets — API keys, tokens, credentials, internal URLs, and exposed GraphQL endpoints. Uses the built-in hunt_js_secrets tool for entropy-based pattern matching across minified bundles.
Covers
Hardcoded SecretsAPI Key Exposure Internal Endpoint LeakageSource Map Exposure
Config Auditor
Infrastructure & Deployment Security
Audits HTTP security headers (CSP, HSTS, X-Frame-Options, etc.), detects exposed admin panels and management interfaces, checks cloud storage buckets for public access, tests for DNS zone transfer vulnerabilities, and probes for open redirect flaws. Leads with a full Nuclei misconfiguration scan.
Covers
HTTP Security HeadersExposed Admin Panels Cloud Storage MisconfigDNS Zone Transfer Open RedirectSubdomain Takeover
Identity Mapper
User Roles & Account Enumeration
Enumerates user roles and privilege levels visible on the application. Tests registration flows, account provisioning processes, username enumeration via timing and error message differences, and enforces username policy checks with boundary values and special characters.

Auth Prober
Authentication & Credential Security
Tests the full authentication attack surface: credential transport over HTTP vs HTTPS, default and weak passwords, account lockout thresholds, auth schema bypasses via header injection, weak password policies, security question flaws, password reset token predictability, alternative channel weaknesses, and MFA presence and bypass vectors.

Authorization & Privilege Escalation
Probes for broken access control: path traversal and file inclusion, direct access to admin-only endpoints without authentication, vertical privilege escalation via role-manipulation payloads, IDOR across user/order/invoice resources, RBAC boundary testing, mass assignment via privileged field injection, and OAuth/OIDC endpoint discovery.

Audits session management end-to-end: session schema analysis, cookie security flags (HttpOnly, Secure, SameSite), fixation attacks, CSRF token presence and SameSite enforcement, logout invalidation, session timeout enforcement, JWT handling including alg:none bypass, session token entropy measurement, and concurrent session policy.

Leads with 9,000+ Nuclei CVE templates, then manually tests reflected XSS, stored XSS, SQL injection (with sqlmap and manual error-detection), server-side code and command injection, SSTI (7*7=49 polyglot), SSRF against cloud metadata endpoints, XXE via XML-accepting endpoints and SOAP, malicious file upload bypass, and file-focused path traversal.
